This project includes a GitHub Actions workflow for automated deployment.

### Fix: "Get Pages site failed"
If you see an error in GitHub Actions stating "Get Pages site failed", follow these steps:
1. Go to your GitHub repository on github.com.
2. Click on **Settings** (top tab).
3. Select **Pages** from the left sidebar.
4. Under **Build and deployment** > **Source**, change the dropdown to **GitHub Actions**.
5. Re-run the failed workflow.
